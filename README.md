i# Ticketbooth

This project is for a workshop demonstration of solidity and smart contract development on the Ethereum blockchain. 
Not suitable for production!

## Installation

Clone the repo, there is no additional setup.

## Usage

The .sol files in the contracts folder are intended to be run on remix.ethereum.org.
